**JAVA PROJECT PANDEMIC**

CAP 1------------------------
**programa : InformacionSobreSilco.java
Función de separar los datos de cada linia de ciudades.txt.

CAP 2------------------------
**programa : DatosYCiudades.java
Mostrar por pantalla y guardar en el fichero ciudadesRedactadas.txt por cada ciudad  sus colindantes y las distancias que las separan.

**programa : ciudades_enfermedad.java
Lee el fichero CCP.bin, genran un nuevo fichero (ciudades_enfermedad.bin) guardando todas las ciudades seguidas del nombre de su enfermedad principal.

**programa : buscando_las_ruta_optimas.java
Dada una ciudad, calcula la ruta más efectiva para llegar a cada una del resto de ciudades,
enumerar las ciudades y generar una matriz de adyacencia, colocando en lugar de 1 la distancia entre las ciudades.
