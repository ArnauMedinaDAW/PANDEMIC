**JAVA PROJECT PANDEMIC**

COMO DESCARGAR:
El repositorio es público, asi que puedes comprimir el proyecto en el boton de <>Code ->Download Zip
o usando github desktop
COMO EJECUTAR:

Abre al paquete de Main, dentro ejecuta la clase Main

IMPORTANTE:
Dentro del proyecto están todas las librerias requeridas, si usas Eclipse IDE, sigue los siguientes pasos:

  1-Abre el proyecto
  2-Clic derecho a JRE System Library
  3-Build Path ->Configure Build Path
  4-Elimina todos los archivos de Modulepath y remplazalos(Add external Jars) con los correspondientes, 
  ubicados dentro de la carpeta del proyecto.
  5-Si se ejecuta fuera del dominio ip del centro Ilerna Lleida, dentro del paquete de Controlador, 
  en la clase ControlDatos, remplaza el contenido 
  de private static final String URL, por el comentario en la linia de arriba.

En caso de querer modificar la interfaz gráfica, instalar en Eclipse IDE la extensión WindowBuilder,
para ver correctamente las imágenes en el editor, deberás cambiar la ruta de las imágenes en las clases de PanelPrincipal y juego, por la ruta absoluta.
